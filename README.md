# FakeStore API Project

## Project Overview
This project covers manual API testing of FakeStoreAPI used by the fictional e-commerce application ShopEase.

## Tools Used
- Postman
- Excel
- GitHub

## APIs Tested
- GET /products
- GET /products/{id}
- GET /products/categories
- GET /products/category/{categoryName}
- POST /carts
- GET /carts/{id}
- GET /users
- GET /users/{id}
- POST /auth/login

## Deliverables
- Test Case Spreadsheet
- Test Execution Report
- Bug / Issue Log
- Test Evidence Screenshots
